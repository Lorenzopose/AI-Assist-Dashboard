# AI Business Intelligence Dashboard

An AI-powered data analysis dashboard that allows users to upload datasets, visualize data, and generate automated insights using Gemini AI.

---

## Overview

This project is a Streamlit-based web application designed to help users quickly explore datasets and extract meaningful insights without deep technical knowledge.

It combines:
- Data visualization
- Statistical summaries
- AI-generated insights
- Natural language querying

---

## Features

### 1. Data Upload
- Supports CSV and Excel files
- Handles errors for invalid formats

### 2. Dashboard
- Key metrics (rows, columns, missing values)
- Data preview
- Summary statistics
- Interactive visualizations

### 3. AI Insights
- Automatically generates:
  - Trends
  - Anomalies
  - Business recommendations

### 4. Ask AI (Chat with Data)
- Users can ask questions about the dataset
- AI provides contextual answers based on data

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- Google Gemini API
